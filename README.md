
# make-mvc-express


## Installation

Install my-project with npx

```bash
  npx make-mvc-express
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env.development file

`PORT`

`MONGODB_URL`
## Run Locally

Make MVC Structure

```bash
  npx make-mvc-express
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
